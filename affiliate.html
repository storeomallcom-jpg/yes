P(q) FORMULA REFERENCE

The Profit Function

// Net Profit over a batch of q units
P(q) = [q · (1 − r) · s]          // Revenue from sold units
     − [q · c]                    // Total purchase cost
     − [q · (1 − r) · a]          // Ad spend (CPA × conversions)
     − [q · r · cret]              // Failed delivery cost
     − Tax(s, q, r)               // Tax on net revenue
     − F                          // Fixed monthly costs

// Break-Even Quantity
BEQ = F ÷ (snet − vc)            // F / contribution per unit
